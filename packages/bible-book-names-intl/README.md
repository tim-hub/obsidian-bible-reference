# Bible Book Names Intl

This is the source of Bible books names in different languages.

- Please help us to add more languages.
- Feel free to submit a PR for it.


## Steps

1. add translation JSON file.
   2. [How to add a translation](src/data/translations/README.md)
3. update index.ts file to include the new language file.
4. update version in package.json and build
5. publish to NPM
